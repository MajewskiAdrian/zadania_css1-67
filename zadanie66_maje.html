1. Pseudoelement :first-line - obejmuje formatowaniem cała pierwszą linie

    2. Przykład:

    p
    {
    color:navy;
    font-size:12pt;
    }

    p:first-line
    {
    color:red;
    font-size:20pt;
    font-weight:bold;
    }

  2. Pseudoelementy: :after oraz :before - oba elementy służą do umieszczania tekstu przed i za formatowanym elementem strony

    3. Przykład:

    p:before
    {
    content:"TO SĄ ĆWICZENIA";
    font-size:15pt;
    font-style:italic;
    font-weight:bold;
    }

    p:after
    {
    content:"AUTOR Król Lew";
    font-size:15pt;
    font-style:italic;
    font-weight:bold;
    }


  3. Pseudoelement :first-letter - pozwala na formatowanie pierwszej litery w bloku tekstu

    4. Przykład:

    p
    {
    color:navy;
    font-size:30pt;
    }

    p:first-letter
    {
    color:red;
    font-size:100pt;
    }
